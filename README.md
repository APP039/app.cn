# 3D Paper Ball Webpage

一个富有哲学意味的3D交互式网页，诠释"先做一个垃圾出来"的设计理念。

## 在线访问

🌐 **GitHub Pages**: https://app039.github.io/app.cn

## 特性

- ✨ 纯黑色背景，营造沉思氛围
- 🗑️ 3D纸张揉团模型，象征创作过程中的不完美
- 🔄 360度自由旋转交互
- 💡 微妙的蓝色光晕效果
- 📱 响应式设计，适配各种设备
- 🎯 极简主义美学，体现原研哉设计风格

## 技术实现

- **Three.js**: 3D渲染引擎
- **WebGL**: 硬件加速图形渲染
- **原生JavaScript**: 交互逻辑
- **CSS3**: 样式和动画效果

## 设计哲学

"先做一个垃圾出来" - 这个理念鼓励创作者：
- 不要追求完美主义而阻碍开始
- 接受创作过程中的不完美
- 通过迭代改进，而非一开始就要求完美
- 每个伟大的作品都始于一个简单的尝试

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/APP039/app.cn.git

# 进入目录
cd app.cn

# 启动本地服务器
python -m http.server 8000

# 访问 http://localhost:8000
```
